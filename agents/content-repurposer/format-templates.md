## What this file is
High-performing content structures per platform. Use as structural reference — match the FORMAT and RHYTHM, never the content.

---

## LinkedIn Template — Technical Release

```
[HOOK LINE 1: Bold factual claim about the tool/release]
[HOOK LINE 2: Setup — what this means for builders]

[BODY BLOCK 1 — What changed]
Paragraph: 3-4 sentences. State the specific change. Add one concrete example.

[BODY BLOCK 2 — Why it matters to you specifically]
Paragraph: 3-4 sentences. Connect to a real problem builders face. Use "if you've ever..."

[BODY BLOCK 3 — What you can actually do with it now]
Paragraph: 3-4 sentences. Specific use case. Numbers or time estimates where possible.

[INSIGHT — The contrarian or non-obvious takeaway]
Single short paragraph. Start with "What most people miss..." or "The surprising part:"

[CTA — Specific question]
"If you're building [specific use case], how are you handling [specific challenge]?"
```

## X Thread Template — Technical Release

```
[1/5] [Hook — must work as standalone tweet, max 220 chars]
[2/5] [Context: what specifically changed, one fact or number]
[3/5] [Practical: what builders can do right now with this]
[4/5] [Mistake: what most people will get wrong about this]
[5/5] [Takeaway: one-sentence summary + implicit CTA]
```

## YouTube Outline Template — Tool Demo

```
THUMBNAIL: [Number] [Thing] [Challenge/Result]
HOOK: [First 15 words — stop mid-thought for curiosity gap]

INTRO:
- Promise: "In this video you'll see exactly how to [outcome]"
- Stakes: "Without this, [problem developers face]"
- Time: "This took me [X hours/days] to figure out"

SECTION 1: [The Problem / Why This Matters]
  - [Specific pain point]
  - [What the old approach cost]

SECTION 2: [The Solution / How It Works]
  - [Core mechanism explained simply]
  - [One concrete demo point]

SECTION 3: [Practical Implementation]
  - [Step or configuration]
  - [Common gotcha]

SECTION 4: [Results / What You Can Build]
  - [Specific outcome]
  - [Next step or extension]

OUTRO: "If this helped, subscribe — I post [cadence] on [topic niche]."
TAGS: [8-10 tags, mix exact + broad]
```

## Instagram Carousel Template — Insight Post

```
CAPTION:
[Hook line — 125 chars max, works as cover text]
[2-3 body lines expanding on the hook]
[1 question to drive saves]
[Hashtags: 3 broad + 5 niche + 2 trending]

SLIDES:
Cover:  [Hook rewritten as visual billboard — 5-8 words]
Slide 2: [Insight title] — [1-2 supporting lines]
Slide 3: [Insight title] — [1-2 supporting lines]
Slide 4: [Insight title] — [1-2 supporting lines]
Slide 5: [Insight title] — [1-2 supporting lines]
CTA:    "Save this if you're building [X]. What would you add?"
```

## TikTok Script Template — Dev Explainer (60s)

```
[0-3s]   HOOK (OVERLAY): [Max 8 words — state the result, not the setup]
[3-10s]  SETUP: [One sentence: why this matters right now for builders]
[10-20s] POINT 1: [8-12 words: concrete fact or step]
[20-30s] POINT 2: [8-12 words: the non-obvious part]
[30-40s] POINT 3: [8-12 words: what to do with this]
[40-50s] PAYOFF: [The surprising conclusion — subverts the setup expectation]
[50-60s] CTA: "Follow for [specific value prop] — [call to action]"
```
